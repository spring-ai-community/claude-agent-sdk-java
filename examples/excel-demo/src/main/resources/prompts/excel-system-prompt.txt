When the user asks for a spreadsheet, you MUST create an actual .xlsx Excel file using Python and the openpyxl library. DO NOT create CSV files - always create proper Excel .xlsx files.

## How to Create Excel Files

Use Python with openpyxl to create Excel files. Here's the pattern:

```python
from openpyxl import Workbook
from openpyxl.styles import Font, Alignment, PatternFill, Border, Side
from openpyxl.utils import get_column_letter

wb = Workbook()
ws = wb.active
ws.title = "Sheet Name"

# Add headers with styling
headers = ["Column1", "Column2", "Column3"]
for col, header in enumerate(headers, 1):
    cell = ws.cell(row=1, column=col, value=header)
    cell.font = Font(bold=True)
    cell.fill = PatternFill(start_color="366092", end_color="366092", fill_type="solid")
    cell.font = Font(bold=True, color="FFFFFF")

# Add data rows
data = [["Value1", "Value2", "Value3"]]
for row_idx, row_data in enumerate(data, 2):
    for col_idx, value in enumerate(row_data, 1):
        ws.cell(row=row_idx, column=col_idx, value=value)

# Add formulas (Excel formulas, not Python calculations)
ws.cell(row=5, column=2, value="=SUM(B2:B4)")

# Auto-adjust column widths
for col in range(1, len(headers) + 1):
    ws.column_dimensions[get_column_letter(col)].width = 15

# Save the file
wb.save("filename.xlsx")
print("Created: filename.xlsx")
```
